# 冒烟测试 — 异常路径

语义理解、防幻觉与兜底逻辑。可在**任意新行程**中穿插验证，不要求走完规划链路。

## 自动化

```bash
uv run pytest backend/tests/test_smoke_flows.py -m smoke_exception -q
```

## 场景表

### E1 城市错别字 + 出发地/目的地分离

| 步骤 | 输入 | 预期 |
|------|------|------|
| 1 | `程度` | 澄清「是否成都？」 |
| 2 | `对` | 仅「目的地：成都」；追问出发城市 |
| 3 | `上海` | 出发=上海，目的地=成都（不重复） |

### E2 口语化假期天数

| 前置 | 输入 | 预期 |
|------|------|------|
| 出发日期 `2026-06-19`（端午节） | `整个假期` | 出行天数=3，继续问人数（不重复同一问句） |

### E3 预算防幻觉

| 前置 | 检查 | 预期 |
|------|------|------|
| 已填目的地/城市/日期/天数，未选预算 | 【当前已确认】 | **无**「一般党 / 2000-5000」 |
| 用户说 `穷游党` | 确认块 | 才出现预算档位 |

### E4 旅行风格防幻觉

| 前置 | 检查 | 预期 |
|------|------|------|
| 仅选预算，未提风格 | 最终确认 | **无** `旅行风格：culture` |

### E5 确认摘要不重复

| 前置 | 检查 | 预期 |
|------|------|------|
| 进入最终需求确认 | 消息体 | 仅一段「我整理一下您的需求」，不与【当前已确认】重复 |

### E6 天数无法理解时的兜底

| 步骤 | 输入 | 预期 |
|------|------|------|
| 1 | 在问天数时发送 `看情况吧` | LLM 辅助或澄清（如「端午节一般是3天，您是指3天吗？」） |
| 2 | — | **不**无限重复「您计划玩几天？」 |

### E7 预算歧义

| 输入 | 预期 |
|------|------|
| `3000左右` | 追问「每人还是总共」 |

### E8 用户纠错

| 输入 | 预期 |
|------|------|
| `不对，改成杭州` | 目的地更新为杭州 |

## Checklist

```
□ E1–E8 任测项行为符合预期
□ pytest -m smoke_exception 全绿
```
