# 上海旅游攻略

## 城市概况

上海是中国经济中心，「魔都」。外滩万国建筑、陆家嘴天际线、石库门里弄与迪士尼并存，适合 3-5 日城市与亲子游。

**最佳旅游季节**：3-5 月、9-11 月；7-8 月炎热，梅雨季 6 月多雨。

**推荐游玩天数**：3-5 天

## 必游景点

### 外滩 + 黄浦江夜游

**简介**：52 栋万国建筑博览，对岸陆家嘴三件套（东方明珠、金茂、上海中心）。

**门票**：外滩免费；夜游约 120-150 元

**游览建议**：15:00-19:00 含日落；可步行至南京东路

### 南京路步行街 + 人民广场

**简介**：中华商业第一街，老字号与百货；人民广场有上海博物馆。

**门票**：免费；上博免费（需预约）

**游览建议**：2-3 小时

### 豫园 + 城隍庙

**简介**：明代江南园林，九曲桥、湖心亭；城隍庙小吃环绕。

**门票**：豫园约 40 元

**游览建议**：上午豫园，中午城隍庙小笼

### 上海迪士尼乐园

**简介**：中国大陆首座迪士尼，适合亲子与年轻人。

**门票**：约 475-799 元（淡旺季）

**游览建议**：全日，提前下官方 App 抢预约；工作日人少

### 陆家嘴 + 东方明珠 / 上海中心

**简介**：登塔俯瞰浦江两岸；上海中心 632 米为中国第一高楼。

**门票**：东方明珠约 199 元起；上海中心观光厅约 180 元

### 田子坊 + 新天地

**简介**：石库门弄堂改造，文艺小店与西餐；新天地为时尚地标。

**门票**：免费

**游览建议**：下午至晚间

### 朱家角 / 七宝古镇

**简介**：江南水乡古镇，距市区约 1 小时。

**门票**：古镇免费，部分景点联票

**游览建议**：半日至一日

## 特色美食

### 必吃清单

1. **小笼包 / 生煎**：南翔馒头店、大壶春、小杨生煎
2. **本帮菜**：老吉士、上海老饭店（响油鳝糊、红烧肉）
3. **蟹粉小笼 / 秃黄油**（秋季）
4. **排骨年糕 / 白切鸡**：鲜得来
5. **蝴蝶酥 / 鲜肉月饼**：国际饭店、光明邨
6. **酒吧与西餐**：新天地、武康路

### 美食街区

- **城隍庙 / 豫园**：游客向小吃
- **云南南路 / 黄河路**：本帮老字号
- **武康路 / 安福路**：咖啡与 brunch

| 外滩 / 南京东路 | 经典地标 | 400-1000 元/晚 |
| 人民广场 | 地铁枢纽 | 350-800 元/晚 |
| 陆家嘴 | 现代江景 | 450-1200 元/晚 |
| 迪士尼周边 | 亲子方便 | 300-700 元/晚 |

## 实用信息

### 交通

- **地铁**：20 条线路覆盖全市，单程 3-7 元
- **机场**：浦东 / 虹桥，磁浮与地铁
- **高铁**：虹桥站 / 上海站

### 温馨提示

1. 迪士尼、上博、豫园旺季预约
2. 外滩节假日人流极大，注意儿童安全
3. 使用交通卡或手机乘车码
4. 夏季防暑，冬季湿冷

## 推荐行程

### 5 日游经典行程

**Day 1**：外滩 → 南京路 → 豫园城隍庙  
**Day 2**：迪士尼乐园全日  
**Day 3**：上博 → 陆家嘴 → 黄浦江夜游  
**Day 4**：田子坊 → 新天地 → 武康路  
**Day 5**：朱家角古镇 → 返程  

**预估花费**（人均，经济型，含迪士尼）：
- 门票：约 700 元
- 住宿：约 1600 元（4 晚）
- 餐饮：约 450 元
- 交通：约 200 元
- **总计**：约 3000-3800 元
