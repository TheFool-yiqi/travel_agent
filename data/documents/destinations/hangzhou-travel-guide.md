# 杭州旅游攻略

## 城市概况

杭州是浙江省省会，南宋古都，西湖列入世界文化遗产。「上有天堂，下有苏杭」，以西湖十景、灵隐禅意、龙井茶与杭帮菜闻名。

**最佳旅游季节**：3-5 月（桃柳烟雨）、9-11 月（秋桂飘香）；夏季较热，梅雨季多雨。

**推荐游玩天数**：3-4 天

## 必游景点

### 西湖风景名胜区（5A，免费）

**简介**：中国最著名的城市湖泊景观，断桥、苏堤、白堤、三潭印月、雷峰塔等。

**门票**：西湖免费；雷峰塔约 40 元；三潭印月船票约 70 元（含岛）

**游览建议**：
- 环湖步行或共享单车，体力有限可乘观光车
- 清晨断桥人少；雷峰塔傍晚登塔看日落
- 音乐喷泉：湖滨银泰附近，晚间场次以公告为准

### 灵隐寺 + 飞来峰（5A）

**简介**：千年古刹，飞来峰石窟造像为江南石窟精华。

**门票**：飞来峰 45 元 + 灵隐寺香花券 30 元（联票）

**游览建议**：上午进香，2-3 小时；可顺访永福寺、韬光寺

### 河坊街 + 南宋御街 + 胡雪岩故居

**简介**：南宋御街改造的历史街区，胡雪岩故居为晚清巨商宅邸。

**门票**：河坊街免费；胡雪岩故居约 20 元

**游览建议**：傍晚至夜间逛吃，2-3 小时

### 龙井村 + 中国茶叶博物馆

**简介**：西湖龙井核心产区，可体验采茶、品茶。

**门票**：茶叶博物馆免费

**游览建议**：半日，春季采茶季最佳

### 西溪国家湿地公园（5A）

**简介**：城市湿地，《非诚勿扰》取景地。

**门票**：约 80 元

**游览建议**：半日至一日，可乘摇橹船

### 九溪烟树 + 云栖竹径

**简介**：西湖西南侧徒步路线，溪水、竹林、茶田。

**门票**：免费或少量收费

**游览建议**：半日徒步，秋季红叶佳

## 特色美食

### 必吃清单

1. **西湖醋鱼**：杭帮菜代表（楼外楼、知味观）
2. **东坡肉**：肥而不腻
3. **叫化鸡**：传统名菜
4. **片儿川**：慧娟面馆、菊英面店
5. **葱包桧**：街头小吃
6. **定胜糕 / 藕粉**：知味观、江南春

### 美食街区

- **河坊街 / 南宋御街**：游客向小吃
- **湖滨 / 南山路**：餐厅集中
- **十五奎巷**：本地小食

## 住宿建议

| 区域 | 优势 | 参考价格 |
|------|------|----------|
| 西湖东侧 / 湖滨 | 近湖景，交通方便 | 300-800 元/晚 |
| 灵隐 / 龙井 | 禅意安静 | 350-900 元/晚 |
| 武林 / 凤起路 | 地铁商业 | 250-600 元/晚 |

## 实用信息

### 交通

- **地铁**：覆盖主要景点，单程 2-7 元
- **高铁**：杭州东 / 杭州西 / 杭州站；上海至杭州约 1 小时
- **机场**：萧山国际机场，机场大巴 + 地铁

### 温馨提示

1. 西湖、灵隐节假日人流大，建议早起
2. 春季备雨具，夏季防晒
3. 杭帮菜偏甜淡，与川菜不同
4. 雷峰塔、灵隐联票可线上购

## 推荐行程

### 3 日游经典行程

**Day 1**：断桥 → 白堤 → 平湖秋月 → 雷峰塔 → 河坊街  
**Day 2**：灵隐寺 → 飞来峰 → 龙井村品茶 → 九溪烟树  
**Day 3**：西溪湿地（或博物馆）→ 湖滨银泰 → 返程  

**预估花费**（人均，经济型）：
- 门票：约 200 元
- 住宿：约 900 元（2 晚）
- 餐饮：约 350 元
- 交通：约 180 元
- **总计**：约 1600-2000 元
