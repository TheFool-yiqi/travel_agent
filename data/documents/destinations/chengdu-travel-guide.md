# 成都旅游攻略

## 城市概况

成都是四川省省会，「天府之国」核心城市，联合国教科文组织「世界美食之都」。以慢生活、川菜、大熊猫和三国文化闻名，适合文化探索与美食之旅。

**最佳旅游季节**：3-5 月、9-11 月（气候舒适）；夏季较热湿，冬季阴冷多雾。

**推荐游玩天数**：4-5 天（市区 2-3 天 + 近郊 1-2 天）

## 必游景点

### 成都大熊猫繁育研究基地（5A）

**简介**：全球最大熊猫保护研究机构之一，百余只大熊猫，上午最活跃。

**门票**：约 55 元（以官方为准）

**游览建议**：
- 建议 7:30 前到达，先看幼年园
- 全程约 3-4 小时，园区较大可乘观光车
- 禁止使用闪光灯

**交通**：地铁 3 号线「动物园站」转景区专线，或打车约 40-60 元

### 武侯祠 + 锦里古街（4A）

**简介**：纪念诸葛亮的祠庙， adjacent 锦里为川西民俗商业步行街。

**门票**：武侯祠约 50 元；锦里免费

**游览建议**：下午逛武侯祠，傍晚锦里夜景与小吃（2-3 小时）

### 宽窄巷子 + 人民公园

**简介**：清代宽窄巷子为老成都缩影；人民公园鹤鸣茶社可体验盖碗茶、掏耳朵。

**门票**：免费

**游览建议**：宽窄巷子 2 小时；人民公园喝茶 1-2 小时

### 都江堰 + 青城山（世界遗产）

**简介**：都江堰为 2200 年水利工程；青城山为道教名山，可联游。

**门票**：都江堰约 80 元；青城山前山约 90 元 + 索道另计

**游览建议**：一日游，上午都江堰、下午青城山；从市区约 1-1.5 小时车程

### 三星堆博物馆（广汉）

**简介**：古蜀文明遗址，青铜面具等文物极具震撼力。

**门票**：约 72 元（需预约）

**游览建议**：半日至一日，距市区约 1 小时；可与熊猫基地分日安排

### 春熙路 / 太古里 / IFS 爬墙熊猫

**简介**：成都现代商业地标，IFS 外墙大熊猫为网红打卡点。

**门票**：免费

## 特色美食

### 必吃清单

1. **四川火锅**：蜀九香、小龙坎等（人均 80-150 元）
2. **串串香**：钢管厂五区小郡肝等（人均 50-80 元）
3. **麻婆豆腐**：陈麻婆豆腐（青羊宫店）
4. **担担面 / 钟水饺 / 龙抄手**：钟水饺、龙抄手总店
5. **夫妻肺片**：夫妻肺片总店
6. **兔头 / 冷锅串串**：玉林、九眼桥一带

### 美食街区

- **锦里 / 宽窄巷子**：游客向，体验一次即可
- **玉林路 / 建设路**：本地夜宵与小吃
- **奎星楼街**：网红与小众餐厅

## 住宿建议

| 区域 | 优势 | 参考价格 |
|------|------|----------|
| 春熙路 / 太古里 | 交通购物便利 | 250-600 元/晚 |
| 宽窄巷子周边 | 老城氛围 | 200-500 元/晚 |
| 武侯祠 / 锦里 | 近景点 | 180-400 元/晚 |

## 实用信息

### 交通

- **地铁**：1-19 号线覆盖主要景点，单程 2-7 元
- **机场**：双流 / 天府国际机场，机场大巴与地铁可达市区
- **高铁**：成都东站、南站连接全国

### 温馨提示

1. 火锅可点鸳鸯锅或微辣，备肠胃药
2. 熊猫基地、三星堆需提前预约
3. 都江堰、青城山穿舒适运动鞋
4. 夏季防晒防蚊，冬季湿冷需保暖

## 推荐行程

### 4 日游经典行程

**Day 1**：熊猫基地 → 宽窄巷子 → 人民公园喝茶 → 火锅  
**Day 2**：武侯祠 → 锦里 → 春熙路 / 太古里  
**Day 3**：都江堰 → 青城山（或三星堆半日 + 市区）  
**Day 4**：东郊记忆 / 博物馆 → 伴手礼（张飞牛肉、蜀绣）→ 返程  

**预估花费**（人均，经济型）：
- 门票：约 350 元
- 住宿：约 800 元（3 晚）
- 餐饮：约 400 元
- 交通：约 250 元
- **总计**：约 1800-2200 元
