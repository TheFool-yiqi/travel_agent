# 北京旅游攻略

## 城市概况

北京是中国首都，六朝古都，世界文化遗产最密集城市之一。故宫、长城、天坛、颐和园代表皇家与京味文化，适合 4-6 日深度游。

**最佳旅游季节**：3-5 月、9-11 月；夏季炎热，冬季干冷但故宫雪景佳。

**推荐游玩天数**：4-5 天

## 必游景点

### 故宫博物院（5A，世界遗产）

**简介**：明清皇宫，世界上现存规模最大、保存最完整的木质古建筑群。

**门票**：约 60 元（旺季），需提前 7 天预约

**开放时间**：08:30-17:00（周一闭馆）

**游览建议**：3-4 小时，中轴线 + 东西六宫；可登景山看全景

### 八达岭 / 慕田峪长城（5A）

**简介**：「不到长城非好汉」，慕田峪相对人少景美。

**门票**：约 40 元 + 索道 / 缆车另计

**游览建议**：一日，慕田峪或八达岭二选一；穿防滑鞋

### 天坛公园（5A，世界遗产）

**简介**：明清皇帝祭天场所，祈年殿、回音壁、圜丘坛。

**门票**：联票约 34 元

**游览建议**：2-3 小时，清晨本地人晨练

### 颐和园（5A，世界遗产）

**简介**：皇家园林，昆明湖、万寿山、长廊、石舫。

**门票**：约 30-60 元（淡旺季）

**游览建议**：半日至一日，可乘船

### 天安门广场 + 国家博物馆

**简介**：世界最大城市广场；国博馆藏丰富。

**门票**：广场免费；国博免费（需预约）

**游览建议**：升旗需查时刻表，安检严格

### 什刹海 + 南锣鼓巷 + 胡同

**简介**：老北京风貌，烟袋斜街、钟鼓楼、四合院。

**门票**：免费（个别院落另计）

**游览建议**：傍晚什刹海酒吧与夜景

### 圆明园 / 清华北大（可选）

**简介**：万园之园遗址；高校需预约参观。

**门票**：圆明园约 25 元

## 特色美食

### 必吃清单

1. **北京烤鸭**：全聚德、便宜坊、四季民福
2. **炸酱面**：海碗居、方砖厂 69 号
3. **涮羊肉**：东来顺、聚宝源
4. **豆汁焦圈**：护国寺小吃
5. **驴打滚 / 豌豆黄 / 艾窝窝**：稻香村、护国寺
6. **卤煮火烧**：北新桥、门框胡同

### 美食街区

- **前门大街 / 大栅栏**：老字号
- **簋街**：夜宵小龙虾
- **牛街**：清真美食

## 实用信息

### 交通

- **地铁**：覆盖主要景点，单程 3-7 元
- **机场**：首都 / 大兴国际机场，机场线
- **高铁**：北京南 / 西 / 朝阳等站

### 温馨提示

1. 故宫、国博、长城等务必提前预约
2. 错峰出行，避开国庆等长假
3. 携带身份证，安检多
4. 冬季防寒，夏季防晒补水

## 推荐行程

### 5 日游经典行程

**Day 1**：天安门 → 故宫 → 景山 → 什刹海  
**Day 2**：慕田峪长城一日游  
**Day 3**：天坛 → 前门大街 → 烤鸭  
**Day 4**：颐和园 → 圆明园（可选）  
**Day 5**：国博 / 798 → 返程  

**预估花费**（人均，经济型）：
- 门票：约 350 元
- 住宿：约 1500 元（4 晚）
- 餐饮：约 500 元
- 交通：约 250 元
- **总计**：约 2600-3200 元
