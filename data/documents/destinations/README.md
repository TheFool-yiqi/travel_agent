## 格式约定

- 每篇含 YAML frontmatter（`city`、`tags`、`sources`、`updated`）
- 正文结构：概况 → 景点 → 美食 → 住宿 → 实用信息 → 行程
- 价格与开放时间仅供参考，规划时应以官方最新信息为准

## 资料来源

内容综合自公开旅游攻略（KKday、Klook、Trip.com、知乎等），经整理改写为项目内部资料，非原文转载。
