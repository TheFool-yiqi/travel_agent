# 厦门旅游攻略

## 城市概况

厦门是福建省副省级城市，「海上花园」「鹭岛」。鼓浪屿为世界文化遗产，兼具闽南文化、海滨风光与文艺气质，适合 3-4 日休闲游。

**最佳旅游季节**：3-5 月、10-12 月；夏季台风季需注意天气。

**推荐游玩天数**：3-4 天（含鼓浪屿 1 日）

## 必游景点

### 鼓浪屿（世界遗产）

**简介**：面积 1.87 平方公里，万国建筑、钢琴之岛，日光岩、菽庄花园、皓月园为标志。

**门票**：上岛船票约 35-50 元（需实名预约）；景点联票约 90 元

**游览建议**：
- 提前 3 天购船票，从邮轮中心或第一码头登岛
- 建议 1 日，避开正午人流
- 日光岩看全景，菽庄花园听涛

### 南普陀寺 + 厦门大学（思明）

**简介**：千年古刹，厦大旁，可登五老峰俯瞰。

**门票**：南普陀免费；厦大需预约参观

**游览建议**：上午进香，2-3 小时

### 环岛路 + 曾厝垵

**简介**：滨海骑行道约 43 公里；曾厝垵为文艺渔村改造街区。

**门票**：免费

**游览建议**：租自行车或电动车，傍晚看日落

### 中山路步行街 + 八市

**简介**：百年骑楼商业街；八市为本地海鲜与小吃市场。

**门票**：免费

**游览建议**：晚间逛中山路，早上去八市

### 集美学村 + 园博苑

**简介**：陈嘉庚建筑群，中西合璧；园博苑展示各地园林。

**门票**：集美免费；园博苑约 60 元

### 福建土楼（南靖 / 永定，一日延伸）

**简介**：世界文化遗产，田螺坑「四菜一汤」、云水谣等。

**门票**：约 90-100 元

**游览建议**：包车或跟团一日，距厦门约 2.5 小时

## 特色美食

### 必吃清单

1. **沙茶面**：乌糖、月华沙茶面
2. **海蛎煎**：莲欢海蛎煎
3. **土笋冻**：闽南特色
4. **姜母鸭**：灌顶姜母鸭
5. **烧肉粽 / 面线糊**：黄则和、1980 烧肉粽
6. **花生汤 / 四果汤**：黄则和花生汤

### 美食街区

- **中山路 / 八市**：本地与游客兼顾
- **曾厝垵**：文艺小吃
- **鼓浪屿**：龙头路小吃

| 思明 / 厦大 | 近南普陀 | 220-480 元/晚 |

## 实用信息

### 交通

- **地铁**：1-3 号线
- **机场**：高崎国际机场，BRT + 地铁
- **高铁**：厦门站 / 厦门北站

### 温馨提示

1. 鼓浪屿船票实名，旺季提前订
2. 夏季防晒防蚊
3. 八市买海鲜注意称重
4. 土楼一日游选正规旅行社

## 推荐行程

### 4 日游经典行程

**Day 1**：中山路 → 八市 → 南普陀 → 厦大外围  
**Day 2**：鼓浪屿全日（日光岩 → 菽庄 → 皓月园）  
**Day 3**：环岛路骑行 → 曾厝垵 → 沙坡尾  
**Day 4**：集美学村（或土楼一日游）→ 返程  

**预估花费**（人均，经济型）：
- 门票与船票：约 200 元
- 住宿：约 900 元（3 晚）
- 餐饮：约 350 元
- 交通：约 180 元
- **总计**：约 1600-2000 元
